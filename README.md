# PuntoX Labs — sitio web

Sitio estático listo para GitHub Pages.

## Publicación rápida
1. Subir `index.html` y la carpeta `assets` al repositorio.
2. Activar GitHub Pages desde Settings > Pages.
3. Source: Deploy from branch.
4. Branch: main.
5. Folder: / root.

## Personalización
- Cambiar el email en `index.html`.
- Cambiar canonical/URL cuando esté definido el dominio.
- Reemplazar el logo tipográfico por el logo final si corresponde.
